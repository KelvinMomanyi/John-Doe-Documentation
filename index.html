The JavaScript libraries have been used to add interactivity to the website. Here's a breakdown of what the code does:

  • 1. Preloader Animation: The preloader animation is removed once the window has fully loaded.
  • 2. Sticky Navigation: The navigation bar becomes sticky (fixed to the top) after scrolling down a certain distance. The `hideNav` function is used to show or hide the navigation bar based on the scroll direction.
  • 3. Typed.js: This library is used to create a typing effect in the header, cycling through different strings like "John Doe", "a Developer", "a Designer", and "a Businessman".
  • 4. Progress Bars: When the section with the ID `exp-id` scrolls into view, the progress bars animate to their respective widths to indicate skill levels.
  • 5. Owl Carousel: This initializes an Owl Carousel for the testimonial section, making it autoplay with a timeout of 4 seconds.
  • 6. Filterizr: This library is used to filter and sort the portfolio items with animation.
  • 7. Navbar Toggle: This code toggles the visibility of the navbar and overlay when the navbar toggler button is clicked.
